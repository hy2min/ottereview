1. 랜딩 페이지 쭉내리기(로그인 확인 필수)
2. 깃허브로 로그인 
3. 레포 연결
   - 어떤 레포?

4. 내가 작성한 PR 목록 필터링 보여주기
   - 내가 작성한 PR
   - 내가 리뷰어인 PR

5. 기존에 있는 레포 들어가서 PR 목록 조회
   - Open, Merge 필터링 해서 보여주기 

6. PR 생성 ( 미리 브랜치 준비 ) 
   - Branch 검증 꼭 develop 으로 !!
   - Convention 체크
   - AI 제목
   - PR 내용 작성
   - 설명 작성하기 (음성, 테스트)
   - PR 생성
   - Github 생성된거 보여주기

7. 코드리뷰
   - 우선 순위 필터링
   - 리뷰 대댓글 남기기
   - 리뷰 남기기(승인)
   - Github에서 코드리뷰 된거 보여주기
   - 머지 보여주기
   - Github에서 머지 된거 보여주기

8. 충돌 페이지
- 충돌 파일 클릭하고 경일이 클릭
- 경일이 email 들어가서 접속
- 코드 공동 편집
   - 1번 내가 수정하고
   - 2번 경일이 수정
- 채팅
- 화이트보드

1. PR 작성할 브랜치 -> PR 생성
